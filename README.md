📘 Caderno Temático – Educação Financeira para Iniciantes
🎯 Objetivos

Este projeto foi desenvolvido utilizando o NotebookLM como ferramenta de aprendizagem ativa.

O objetivo foi reunir materiais confiáveis sobre educação financeira e utilizá-los para construir um miniguia de estudos, explorando técnicas de engenharia de prompts para melhorar a qualidade das respostas geradas pela IA.

Ao final do estudo, espera-se compreender:

como organizar um orçamento pessoal;
importância da reserva de emergência;
diferenças entre renda fixa e renda variável;
funcionamento dos juros compostos;
conceitos básicos para começar a investir.
