# 📖 Sobre o Projeto

Este projeto foi desenvolvido utilizando o **Google NotebookLM** para organizar fontes de estudo, sintetizar conteúdos e construir um caderno temático sobre **Educação Financeira para Iniciantes**.

Durante o desenvolvimento foram utilizadas diversas fontes abertas, explorando técnicas de **Engenharia de Prompts** para obter respostas mais completas, organizadas e contextualizadas.

O resultado é um **miniguia de estudos** contendo resumos, glossário e prompts reutilizáveis para futuras revisões.

---

# 🎯 Objetivos

- Compreender os fundamentos da Educação Financeira.
- Aprender como elaborar um orçamento pessoal.
- Entender a importância da reserva de emergência.
- Diferenciar investimentos de renda fixa e renda variável.
- Compreender o funcionamento dos juros compostos.
- Utilizar Inteligência Artificial como ferramenta de aprendizagem ativa.

# 🧠 Engenharia de Prompts

Durante o projeto foram testadas diferentes abordagens para extrair respostas mais completas da IA.

## Prompt 1

> Explique educação financeira para uma pessoa que nunca investiu.

**Resultado**

Foi gerada uma explicação simples, utilizando exemplos do cotidiano.

---

## Prompt 2

> Compare renda fixa e renda variável em formato de tabela.

**Resultado**

A resposta apresentou uma comparação clara entre risco, rentabilidade, previsibilidade e exemplos de investimentos.

---

## Prompt 3

> Explique juros compostos utilizando exemplos práticos.

**Resultado**

A IA apresentou simulações numéricas que facilitaram a compreensão do conceito.

---

## Prompt 4

> Crie um mapa mental sobre Educação Financeira.

**Resultado**

Foi produzido um mapa mental organizado contendo os principais tópicos estudados.

---

## Prompt 5

> Gere cinco questões de revisão sobre Educação Financeira.

**Resultado**

Foram criadas perguntas objetivas úteis para revisão do conteúdo.

# ⚠️ Dificuldades Encontradas

Durante os testes foi possível observar que:

- Alguns prompts geravam respostas muito genéricas.
- Foi necessário especificar o nível de conhecimento esperado.
- Solicitar exemplos práticos melhorou significativamente a qualidade das respostas.
- Pedir respostas em formato de tabela ou tópicos tornou o conteúdo mais organizado.

# 📘 Miniguia de Estudos

## 💰 Educação Financeira

Consiste no conjunto de conhecimentos que permite administrar melhor o dinheiro, planejar gastos e tomar decisões conscientes.

---

## 📊 Orçamento Pessoal

É o planejamento das receitas e despesas com o objetivo de manter equilíbrio financeiro.

---

## 🚨 Reserva de Emergência

Valor reservado para cobrir imprevistos, evitando endividamentos.

O recomendado é possuir uma reserva equivalente a **3 a 6 meses das despesas mensais**.

---

## 📈 Juros Compostos

São juros calculados sobre o valor inicial acrescido dos juros acumulados anteriormente.

São conhecidos como **"juros sobre juros"**.

---

## 🏦 Renda Fixa

Características:

- Maior previsibilidade
- Menor risco
- Indicada para objetivos de curto e médio prazo

Exemplos:

- Tesouro Selic
- CDB
- LCI
- LCA

---

## 📉 Renda Variável

Características:

- Maior risco
- Oscilações de mercado
- Potencial de maior rentabilidade

Exemplos:

- Ações
- ETFs
- Fundos Imobiliários (FIIs)

# 📖 Glossário

| Conceito | Definição |
|----------|-----------|
| Patrimônio | Conjunto de bens e direitos |
| Liquidez | Facilidade para transformar um investimento em dinheiro |
| Rentabilidade | Ganho obtido em um investimento |
| CDI | Taxa utilizada como referência para diversos investimentos |
| Selic | Taxa básica de juros da economia brasileira |
| Inflação | Aumento generalizado dos preços |
| Diversificação | Distribuição dos investimentos para reduzir riscos |
